# E-commerce Sales & RFM Customer Segmentation

## 📌 Project Overview
Комплексный анализ транзакций онлайн-ритейлера (500k+ строк). Проект включает в себя очистку данных, исследовательский анализ (EDA), когортный анализ и сегментацию клиентов по методике RFM.

## 🛠 Стек технологий
* **Python** (Pandas, Matplotlib, Seaborn) — обработка и анализ.
* **SQL** — Извлечение данных из базы, фильтрация и агрегация (GROUP BY, HAVING).
* **RFM-анализ** — сегментация базы на 10 групп.
* **DataLens** — интерактивный дашборд для бизнеса.

## 🚀 Ключевые этапы
1. **Data Cleaning:** Удаление аномалий, обработка возвратов и технических транзакций.
2. **EDA:** Анализ выручки и географии продаж.

7. **BI Dashboard:** Создание интерактивной витрины данных.

## 📊 Результаты
* Выявлено ядро лояльных клиентов (VIP), приносящее более 50% выручки.
* Определена "Зона риска" — сегменты клиентов, требующих немедленной реактивации.
